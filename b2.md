# Dockerizing MySQL Database and Demonstrating Client-Server Model Using Docker

# Aim
To create a Docker container for MySQL database and demonstrate the working of the client-server architecture using Docker containers.

---

# Prerequisites

Before starting, install:

- Docker Desktop
- MySQL Client (optional)

Official Docker Website:
https://www.docker.com/products/docker-desktop/

---

# Step 1: Verify Docker Installation

Open terminal and run:

```bash
docker --version
```

Expected Output:

```bash
Docker version 27.x.x
```

---

# Step 2: Pull MySQL Docker Image

Download MySQL image from Docker Hub.

```bash
docker pull mysql:8.0
```

---

# Step 3: Verify Downloaded Image

```bash
docker images
```

Expected Output:

```bash
REPOSITORY   TAG       IMAGE ID       CREATED       SIZE
mysql        8.0       abc12345       2 days ago    600MB
```

---

# Step 4: Create MySQL Server Container

Run the MySQL server container.

```bash
docker run --name mysql-server \
-e MYSQL_ROOT_PASSWORD=root123 \
-p 3306:3306 \
-d mysql:8.0
```

---

# Explanation of Command

| Option | Description |
|---|---|
| --name mysql-server | Container name |
| -e MYSQL_ROOT_PASSWORD=root123 | Sets MySQL root password |
| -p 3306:3306 | Maps MySQL port |
| -d | Runs container in background |
| mysql:8.0 | MySQL image |

---

# Step 5: Verify Running Container

```bash
docker ps
```

Expected Output:

```bash
CONTAINER ID   IMAGE       STATUS       PORTS                    NAMES
abcd1234       mysql:8.0   Up 2 mins    0.0.0.0:3306->3306/tcp   mysql-server
```

---

# Step 6: Access MySQL Server Container

Open MySQL terminal inside container.

```bash
docker exec -it mysql-server mysql -u root -p
```

Enter password:

```bash
root123
```

---

# Step 7: Create Database

Inside MySQL shell:

```sql
CREATE DATABASE studentdb;
```

Verify databases:

```sql
SHOW DATABASES;
```

---

# Step 8: Create Table

Use database:

```sql
USE studentdb;
```

Create table:

```sql
CREATE TABLE students(
    id INT PRIMARY KEY,
    name VARCHAR(50),
    department VARCHAR(50)
);
```

---

# Step 9: Insert Data

```sql
INSERT INTO students VALUES

(2,'Rahul','ISE');
```

# Step 11: Demonstrate Client-Server Model Using Docker

## Server Container
The MySQL container acts as the database server.

Container:
```bash
mysql-server
```

---

## Client Container
Create another container as MySQL client.

```bash
docker run -it --network host mysql:8.0 mysql -h 127.0.0.1 -u root -p
```

Enter password:

```bash
root123
```

---

# Client-Server Working

## Server Responsibilities
- Stores database
- Processes queries
- Returns results

## Client Responsibilities
- Sends SQL requests
- Receives data from server
