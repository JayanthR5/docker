# Jenkins CI/CD – Slack & Email Notification Integration

## Objective

Configure Jenkins to send:

* Slack notifications for build status
* Email notifications for CI/CD pipeline results

This guide provides a clean step-by-step setup for beginners.

---

| Tool                                                          | Purpose                   |
| ------------------------------------------------------------- | ------------------------- |
| [Jenkins](https://www.jenkins.io/?utm_source=chatgpt.com)     | Automation Server         |
| [Slack](https://slack.com/intl/en-in/?utm_source=chatgpt.com) | Team Communication        |
| Gmail SMTP                                                    | Email Notifications       |


# Jenkinsfile with Slack & Email Notifications

```groovy
pipeline {
    agent any

    stages {

        stage('Build') {
            steps {
                bat 'echo Building Application...'
            }
        }

        stage('Test') {
            steps {
                bat 'echo Running Tests...'
            }
        }

        stage('Deploy') {
            steps {
                bat 'echo Deploying Application...'
            }
        }
    }

    post {

        started {
            slackSend(
                channel: '#devops-group-channel',
                message: "🚀 Build Started: ${env.JOB_NAME} #${env.BUILD_NUMBER}"
            )
        }

        success {

            slackSend(
                channel: '#devops-group-channel',
                message: "✅ Build SUCCESS: ${env.JOB_NAME} #${env.BUILD_NUMBER}"
            )

            emailext(
                subject: "SUCCESS: ${env.JOB_NAME} #${env.BUILD_NUMBER}",
                body: """
Build completed successfully.

Job Name: ${env.JOB_NAME}
Build Number: ${env.BUILD_NUMBER}

Build URL:
${env.BUILD_URL}
""",
                to: "team@example.com"
            )
        }

        failure {

            slackSend(
                channel: '#devops-group-channel',
                message: "❌ Build FAILED: ${env.JOB_NAME} #${env.BUILD_NUMBER}"
            )

            emailext(
                subject: "FAILED: ${env.JOB_NAME} #${env.BUILD_NUMBER}",
                body: """
Build failed.

Job Name: ${env.JOB_NAME}
Build Number: ${env.BUILD_NUMBER}

Check Console Output:
${env.BUILD_URL}
""",
                to: "team@example.com"
            )
        }
    }
}
```

# Common Errors and Solutions

| Error                        | Solution                           |
| ---------------------------- | ---------------------------------- |
| Slack notification failed    | Verify Slack token and channel     |
| Email not sending            | Check SMTP server and app password |
| Invalid credentials          | Re-enter credentials               |
| Jenkins pipeline failed      | Check Console Output               |
| Slack test connection failed | Verify workspace name              |
