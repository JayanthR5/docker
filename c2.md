# Jenkins CI/CD – Slack & Email Notification Integration

## Objective

Configure Jenkins to send:

* Slack notifications for build status
* Email notifications for CI/CD pipeline results

This guide provides a clean step-by-step setup for beginners.

---

# Tools Required

| Tool                                                          | Purpose                   |
| ------------------------------------------------------------- | ------------------------- |
| [Jenkins](https://www.jenkins.io/?utm_source=chatgpt.com)     | Automation Server         |
| [Slack](https://slack.com/intl/en-in/?utm_source=chatgpt.com) | Team Communication        |
| Gmail SMTP                                                    | Email Notifications       |
| Slack Notification Plugin                                     | Jenkins Slack Integration |
| Email Extension Plugin                                        | Jenkins Email Alerts      |

---



---

# Step 1: Open Jenkins Dashboard

Open browser:

```text
http://localhost:8080
```

Login with Jenkins admin credentials.

---

# Step 2: Install Required Plugins

Go to:

```text
Manage Jenkins
→ Plugins
→ Available Plugins
```

Search and install:

```text
Slack Notification Plugin
Email Extension Plugin
```

After installation:

```text
Restart Jenkins
```

---

# Step 3: Create Slack Channel

Open Slack:

[Slack Official Website](https://slack.com/intl/en-in/?utm_source=chatgpt.com)

Create a channel:

```text
#devops-group-channel
```

---

# Step 4: Configure Slack Integration

Open Slack Jenkins Integration:

[Slack Jenkins CI Integration](https://my.slack.com/services/new/jenkins-ci?utm_source=chatgpt.com)

---

## Configure Integration

### Select Channel

Choose:

```text
#devops-group-channel
```

Click:

```text
Add Jenkins CI Integration
```

---

## Copy Credential

Slack provides:

```text
Secret Token / Credential ID
```

Copy this token.

---

# Step 5: Add Slack Credentials in Jenkins

Go to:

```text
Manage Jenkins
→ Credentials
→ System
→ Global Credentials
→ Add Credentials
```

Select:

| Field  | Value             |
| ------ | ----------------- |
| Kind   | Secret Text       |
| Secret | Paste Slack Token |
| ID     | slack-token       |

Click:

```text
Create
```

---

# Step 6: Configure Slack in Jenkins

Go to:

```text
Manage Jenkins
→ Configure System
```

Find:

```text
Slack Section
```

Add:

| Field           | Value                 |
| --------------- | --------------------- |
| Workspace       | your-workspace-name   |
| Credential      | slack-token           |
| Default Channel | #devops-group-channel |

---

## Test Slack Connection

Click:

```text
Test Connection
```

### Expected Output

```text
Success
```

Slack channel receives:

```text
Jenkins connected successfully
```
