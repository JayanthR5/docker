# Docker Setup and Deployment Guide

## Objective
Create a Docker image for an application stored in a local repository and run the application using Docker.

---

# Step 1: Install Docker

## For Windows

### Download Docker Desktop
Official Website:
https://www.docker.com/products/docker-desktop/

### Installation Steps
1. Download Docker Desktop.
2. Run the installer.
3. Enable:
   - WSL 2
   - Virtual Machine Platform
4. Restart the system.
5. Open Docker Desktop.
6. Verify installation:

```bash
docker --version
```

Expected Output:

```bash
Docker version 27.x.x
```

---

# Step 2: Verify Docker Installation

Run:

```bash
docker run hello-world
```

If Docker is installed correctly, a success message will appear.

---

# Step 3: Project Structure

```bash
myapp/
```

---

# Step 4: Create Sample Application

## app.py

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Docker Application Running Successfully!"

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```

---

## requirements.txt

```txt
flask
```

---

# Step 5: Create Dockerfile

Create a file named:

```bash
Dockerfile
```

Add the following code:

```dockerfile
# Base Image
FROM python:3.11

# Set working directory
WORKDIR /app

# Copy files
COPY . .

# Install dependencies
RUN pip install -r requirements.txt

# Expose port
EXPOSE 5000

# Run application
CMD ["python", "app.py"]
```

---

# Step 6: Build Docker Image

Open terminal inside project folder.

Run:

```bash
docker build -t myapp .
```

### Explanation
- `docker build` → Builds image
- `-t myapp` → Image name
- `.` → Current directory

---

# Step 7: Verify Docker Image

Run:

```bash
docker images
```

Example Output:

```bash
REPOSITORY   TAG       IMAGE ID       CREATED       SIZE
myapp        latest    abc12345       1 minute ago  300MB
```

---

# Step 8: Run Docker Container

Run:

```bash
docker run -d -p 5000:5000 --name mycontainer myapp
```

### Explanation
- `-d` → Detached mode
- `-p 5000:5000` → Port mapping
- `--name mycontainer` → Container name

---

# Step 9: Verify Running Container

Run:

```bash
docker ps
```

---

```bash
http://localhost:5000
```

Output:


# Step 13: Remove Docker Image

```bash
docker rmi myapp
```

---

# Important Docker Commands

| Command | Description |
|---|---|
| docker images | Show images |
| docker ps | Running containers |
| docker ps -a | All containers |
| docker stop container_name | Stop container |
| docker start container_name | Start container |
| docker rm container_name | Remove container |
| docker rmi image_name | Remove image |

# Advantages of Docker

- Easy deployment
- Platform independent
- Lightweight virtualization
- Faster application setup
- Consistent development environment

---

# Conclusion

Using Docker:
1. The application is packaged inside an image.
2. The image is used to create containers.
3. Containers run consistently on any system with Docker installed.

This process simplifies deployment and application management.
